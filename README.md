# CBMapping

**CBMapping** — Python-веб-проект для интерактивного картографического сервиса с серверной частью и ботом для автоматизации задач.  

Проект включает:

## Структура репозитория

- `bot.py` — основной скрипт запуска Discord бота.  
- `site.py` — основной серверный модуль для обработки HTTP-запросов.  
- `servers/` — вспомогательные серверные модули.
- `templates/` — HTML-шаблоны веб-интерфейса.  
- `static/` — статические ресурсы (CSS, JS, изображения).  
- `admin_static/` — статические ресурсы для админской панели.  
- `pages_data/` — данные и конфигурации для страниц.  
- `requirements.txt` — зависимости Python.  
- `run.ps1` — скрипт для запуска на Windows через PowerShell всей системы.  

## Основные функции

- Веб-сервер на Python для обработки запросов и отображения страниц.  
- Веб-интерфейс с шаблонами и статикой.  
- Интеграция с ботом для автоматизации операций и взаимодействия с пользователями.  
- Модульная структура, позволяющая легко расширять функционал.

## Предназначение

Проект служит платформой для интерактивной визуализации данных, управления картографическим сервисом через веб-интерфейс и автоматизации задач с помощью бота.  

---

**Ключевые технологии**: Python, HTTP-сервер, HTML-шаблоны, статические ресурсы, боты (Discord).
